# Node.js API Project

## Features
- Multiple API endpoints
- Request logging  
- Error handling
- Visitor counter

## Endpoints
- `/` - Home page with visitor count
- `/api/Abdullahi` - Personal information
- `/api/contact` - Contact details

## How to Run
1. Install dependencies: `npm install`
2. Start server: `node app.js`
3. Visit: `http://localhost:3000`
